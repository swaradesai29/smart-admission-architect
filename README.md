# Smart Admission Architect

A data-driven MHT-CET college predictor that helps engineering aspirants find optimal colleges using:

- Cutoff trend prediction
- ROI analysis
- Placement data
- Admission probability scoring

## Tech Stack

Frontend
- HTML
- TailwindCSS
- JavaScript
- Chart.js

Backend
- Python
- Pandas

## Features

- Predict admission chances
- Compare colleges
- ROI ranking
- Cutoff trend visualization
